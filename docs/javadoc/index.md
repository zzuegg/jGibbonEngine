## Generating docs locally

The project uses **Dokka** for HTML documentation generation. To build the API docs locally:

```bash
# Clone the repository
git clone https://github.com/zzuegg/jGibbonEngine.git
cd jGibbonEngine

# Generate Dokka HTML docs
./gradlew dokkaHtml

# Open in browser
open build/dokka/html/index.html   # macOS
xdg-open build/dokka/html/index.html  # Linux
```

The output is placed in `build/dokka/html/`.

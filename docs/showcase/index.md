## What to include in your submission

When opening a showcase issue, please provide:

- **Project name** and short description (1–2 sentences)
- **Category** — game, tool, demo, tech demo, library
- **GitHub / itch.io / website link**
- **Screenshot or GIF** — at least 800 × 450 px
- **jGibbonEngine version** used
